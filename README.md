## Description

**Fashion Park** is a modern, responsive, and high-performance e-commerce platform frontend designed to offer a seamless online shopping experience, with a specific focus on showcasing a collection of **fashionable T-shirts**.

Built using **React** and styled with **Tailwind CSS**, this project demonstrates a comprehensive frontend solution for a niche e-commerce market, featuring dynamic product listings, detailed item views, and intuitive navigation system.

---

## Live Demo

Experience the live application here:

## Features

-   **T-shirt Focused Catalog:** Browse a wide and curated range of **fashionable T-shirts** with high-quality, zoomable images.
-   **Fully Responsive Design:** Optimized layout for all screen sizes (desktop, tablet, and mobile) ensures a consistent user experience.
-   **Product Detail View:** Comprehensive page for individual T-shirt products, showing specifications, sizing charts, and pricing.
-   **Shopping Cart Functionality:** Allows users to add, remove, and manage selected items before checkout.
-   **Modern UI/UX:** Clean, elegant, and modern design powered by the utility-first framework **Tailwind CSS**.

## Getting Started

Follow these instructions to get a local copy of the project up and running.

### Prerequisites

You need to have **Node.js** and **npm** (or **yarn**) installed on your machine.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Tanvirho/FASHION-PARK.git
    cd fashion-park
    ```
2.  **Install project dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

### Running the Application

Start the local development server:

```bash
npm start
# or
yarn start
